## 应用日志

### HTTP access.log

```
127.0.0.1 - - [07/Jun/2025:10:00:00 +0000] "GET /index.html HTTP/1.1" 200 2326
```

- `127.0.0.1`：客户端 IP 地址
- `[07/Jun/2025:10:00:00 +0000]`：请求时间戳
- `"GET /index.html HTTP/1.1"`：请求方法、请求的 URL 和 HTTP 版本
- `200`：返回的 HTTP 状态码
- `2326`：返回的字节数
